# MineClone - forsenE Edition

A comprehensive 2D Minecraft-inspired survival game built as an Odoo MCP Studio web application, featuring Forsen/Twitch meme theming and Odoo references.

## Features

- Procedurally generated infinite world with chunk-based loading
- Mining and block placement mechanics
- Multiple block types with unique properties (ores, explosives, terrain)
- Weapon bonuses from mining rare ores (damage, firerate, speed, multishot)
- Gun combat with upgradeable stats
- Enemy AI with multiple enemy types (Pepega, monkaS, forsenE, Odoo Bug, MADMONQ Enjoyer)
- Boss battles:
  - MEGALUL Horse (charge attack)
  - Sebastian forsenPls (teleportation)
  - forsenClown (erratic movement)
  - FROG forsenFrog (leap and tongue attack)
  - ENDER DRAGON forsenInsane (flying, fire breath, 3 phases)
- World powerup drops (health, damage, shield, nuke, mega gun, block breaker)
- Block breaker powerup - bullets destroy blocks
- Progressive difficulty scaling over time
- Persistent world saves with user storage
- Custom procedural sound effects and background music
- Forsen/Twitch meme death messages, kill messages, and splash texts
- Odoo-themed block names and tips

## Controls

- **WASD / Arrow Keys** - Move
- **Space** - Jump
- **Left Click** - Dig blocks / Shoot gun
- **Right Click** - Place blocks
- **1-9** - Select hotbar slot
- **ESC** - Pause menu

## Gameplay

- Mine blocks to collect resources
- Rare ores (diamond, emerald, gold, iron) have a chance to grant permanent weapon upgrades
- Enemies spawn and get stronger over time
- Collect world powerups for temporary boosts
- Boss enemies spawn after certain time thresholds and drop bonus upgrades
- Survive as long as possible and rack up a high score

## Import

1. Download `MineCloneforsenE_Edition.csv` from this folder
2. In Odoo, go to **MCP Server > Web Apps** list view
3. Import the `.csv` file using the standard Odoo import function

## Requirements

- [Odoo MCP Studio](https://apps.odoo.com/apps/modules/19.0/odoo_remote_mcp)
