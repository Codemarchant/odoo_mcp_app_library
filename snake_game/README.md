# Snake Game

A classic Snake game built as an Odoo MCP Studio web application with persistent user stat tracking.

## Features

- Classic snake gameplay with keyboard arrow controls
- Score tracking during gameplay
- Persistent user statistics (high score, games played, total score) stored per user
- Responsive design with Tailwind CSS
- has custom sound effects, custom background changes

## Import

1. Download `snake_game.csv` from this folder
2. In Odoo, go to **MCP Server > Web Apps** list view
3. Import the `.csv` file using the standard Odoo import function

## Requirements

- [Odoo MCP Studio](https://apps.odoo.com/apps/modules/19.0/odoo_remote_mcp)
